## ROS2 Installation

Für Ubuntu Noble (24.04) gibt es Deb Pakete für ROS2 Jazzy Jalisco. Noble und Jazzy Jalisco sind Namen für die jeweilige Version.

Zuerst gehen wir sicher, dass unser Ubuntu System UTF-8 nutzt.

Mit <kbd>STRG</kbd> + <kbd>ALT</kbd> + <kbd>T</kbd> rufen wir ein neues Terminal auf.

Nun geben wir `locale` ein und prüfen, ob unser System mit UTF-8 arbeitet.
